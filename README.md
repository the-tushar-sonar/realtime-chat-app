# Realtime Chat Application

## Tech Stack
- React
- Node.js + Express
- Socket.IO
- MongoDB
- FastAPI (Python)
- Docker

## Features
- Realtime messaging
- Room-based chat
- Toxic message detection using ML
- Dockerized microservices architecture

## Setup
```bash
docker compose up --build
```
## Architecture
- Frontend (React)
- Backend (Node.js)
- ML Service (FastAPI)
- Database (MongoDB)
